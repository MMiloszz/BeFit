Projekt zaliczeniowy wykonany w ASP.NET Core MVC.
Aplikacja służy do zapisywania treningów użytkownika, dodawania wykonanych ćwiczeń
oraz przeglądania statystyk z ostatnich 4 tygodni.
W projekcie wykorzystano Entity Framework Core oraz ASP.NET Identity.
Administrator zarządza typami ćwiczeń, a zalogowany użytkownik
może tworzyć tylko własne sesje treningowe i ćwiczenia.
